Вам нужно написать программу, которая будет читать файл, заниматься его чисткой, формировать почтовые адреса для сотрудников, генерировать пароли безопасности для входа в почту, заносить информацию в этот файл и перезаписывать его.
1. Ваша программа должна очистить файл от всех невалидных данных (строки с пустыми данными, некорректными именами, некорректными телефонами и городами)
2. При очистке исходного файла, все невалидные данные должны удаляться из него и сохраняться в отдельный текстовый файл
3. Поиск информации в файле должен осуществляться с помощью регулярных выражений.
4. Напишите свою функцию формирования почтового адреса.
5. Сгенерированные пароли должны заноситься в исходный файл в отдельный столбец "PASSWORD".
